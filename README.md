## Using Equations in Google Docs (Chrome Fix)

### Setup
If the **Equation** option is missing in Google Docs:

1. Open your document in **Google Chrome**
2. Click **Help → Search the menus** (or press `Option + /`)
3. Type: `equation`
4. Click **“Show equation toolbar”**

This will enable the equation editor (π icon), allowing you to insert and format mathematical expressions.

### Usage
- Click **New equation** at the top of the page  
- This will activate the equation editor so you can type math expressions
